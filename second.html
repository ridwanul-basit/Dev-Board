Question-1: What are the different ways to select an element in the DOM?

We can select an element in DOM by using getElementById("id") , getElementsByClassName("class") , getElementsByTagName("tag") , querySelector("selector") , querySelectorAll("selector").

Question-3: What is event delegation in the DOM?

Event delegation is a technique where instead of attaching event listeners to individual child elements.This improves performance and simplifies code when dealing with dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event starts from the target element and bubbles up through its parent elements until it reaches the root of the document.This allows the event to be handled at multiple levels of the DOM, which is useful for event delegation.
